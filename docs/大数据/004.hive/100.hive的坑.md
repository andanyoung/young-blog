很明显是用户权限的问题。

解决方案：

为 user 添加权限，`hadoop fs -chmod 777 /user`

创建数据库是加在有权限的目录下面（修改 hdfs 的权限 `hadoop fs -chown xxx:xxx /user/hive/xxx`）

```
hive (default)> create database db_hive2 location '/db_hive2';
```
